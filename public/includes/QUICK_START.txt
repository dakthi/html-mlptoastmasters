QUICK START GUIDE - MLP Toastmasters Navigation & Footer
================================================================================

GET NAVIGATION & FOOTER ON YOUR PAGE IN 30 SECONDS
================================================================================

STEP 1: Copy this line
────────────────────────────────────────────────────────────────────────────
<script src="/js/include-nav-footer.js" defer></script>
────────────────────────────────────────────────────────────────────────────

STEP 2: Paste it in your HTML <head> section
────────────────────────────────────────────────────────────────────────────
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Page Title</title>

    <!-- Your other links here -->
    <script src="https://cdn.tailwindcss.com"></script>

    <!-- ADD THIS LINE: -->
    <script src="/js/include-nav-footer.js" defer></script>

</head>
<body>
    <!-- Navigation will appear here automatically -->

    <main>
        <!-- Your page content here -->
    </main>

    <!-- Footer will appear here automatically -->
</body>
</html>
────────────────────────────────────────────────────────────────────────────

STEP 3: Done!
Your page now has:
✓ Responsive navigation with logo
✓ Mobile hamburger menu
✓ Search functionality
✓ Professional footer with links
✓ Auto-updating copyright year

================================================================================
CUSTOMIZATION (Later)
================================================================================

To customize, edit these files:

1. Change Colors:
   File: /includes/nav.html and /includes/footer.html
   Find: "text-blue-900"
   Replace with: "text-red-900" (or your color)

2. Update Navigation Links:
   File: /includes/nav.html
   Change the <a> tags in the desktop and mobile nav sections

3. Update Footer Content:
   File: /includes/footer.html
   Update contact info, links, schedule, social media

4. Advanced Search:
   File: /js/include-nav-footer.js
   Modify the performSearch() method

See README.md and INTEGRATION_GUIDE.md for detailed instructions.

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Navigation and footer don't appear
Answer: Make sure the script tag is in the <head> section, not the <body>

Problem: Page looks broken
Answer: Make sure Tailwind CSS CDN is loaded before the script tag

Problem: Mobile menu doesn't work
Answer: Clear your browser cache and refresh the page

Problem: Search doesn't work
Answer: Check that /blog_posts.json exists with proper structure

For more help, see: README.md in this directory
