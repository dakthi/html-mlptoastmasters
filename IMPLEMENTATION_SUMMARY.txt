================================================================================
HOW TO USE
================================================================================

BASIC IMPLEMENTATION (One line per page):

1. Open your HTML file
2. Add to the <head> section (after Tailwind CSS):
   <script src="/js/include-nav-footer.js" defer></script>
3. Save the file
4. Test on mobile and desktop

THAT'S IT!

The navigation will automatically appear at the top of your page.
The footer will automatically appear at the bottom of your page.

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Navigation/footer not appearing
Solution:
  1. Check script tag is in <head> (not <body>)
  2. Verify path is /js/include-nav-footer.js
  3. Check F12 console for errors
  4. Ensure Tailwind CSS is loaded

Issue: Search not working
Solution:
  1. Check /blog_posts.json exists
  2. Verify JSON structure is correct
  3. Check F12 Network tab for fetch errors
  4. Ensure search query is 2+ characters

Issue: Mobile menu stuck
Solution:
  1. Clear browser cache
  2. Hard refresh (Ctrl+Shift+R)
  3. Check F12 for JavaScript errors
  4. Test in different browser

Issue: Styling issues
Solution:
  1. Clear browser cache
  2. Verify Tailwind CSS is loaded
  3. Check for CSS conflicts
  4. Inspect element in DevTools

See /public/includes/README.md for more troubleshooting.
